InternalError Erstellt eine Instanz, die einen Fehler repräsentiert, der auftritt, wenn ein interner Fehler in JavaScript auftaucht (z. B. zu viel Rekursion).

EvalErrorErstellt eine Instanz, die einen Fehler repräsentiert, der bei der globalen eval() Funktion auftritt.

ReferenceErrorErstellt eine Instanz, die einen Fehler repräsentiert, der auftritt, wenn eine nicht valide Referenz referenziert werden soll.
